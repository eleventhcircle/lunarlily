Helpful resources

Various resources, tutorials, guides, and Cool Things that I've collected while cruising around amongst the stars, loosely grouped into categories for easier browsing.

Web dev & tools

  • Zonelots

    A blogging/journal framework based on the popular Zonelets that adds additional functionality like tags. Makes use of Javascript to organize and index entries.

  • File Garden

    A free file hosting service - very helpful for managing the size of your site as it lets you offload images (generally the biggest files)

  • JSONLint

    A free format validator specifically for JSON. Invaluable for double-checking escaped characters if you happen to store HTML segments inside of JSON objects (like me!)

  • W3C Feed Validator

    Free tool to check for syntax errors in RSS and Atom feeds. Genuine lifesaver, if you bookmark nothing else please bookmark this.

  • Google Fonts

    Massive resource for open-source and free-to-use fonts, you can either download + self-host font files or hook them into your site's CSS via their API.

Graphics & design

  • Photopea

    An browser-based image editor, a la Photoshop

  • ezgif

    gif maker/editor

  • resizepixel

    A different online image editor than Photopea, simpler UI with less editing options. Good for quick image crops and whatnot!

  • ditherit

    Image dithering tool

  • Wada Sanzo's Dictionary of Color Combinations

    An online collection of curated color palettes and studies by Wada Sanzo, a renowned Japanese artist and costume designer

  • coolers.co

    Useful color palette tool, able to colorpick palette from images which I find particularly useful

  • GifCities

    Archive of graphic media salvaged from Geocities (really cool project from the Internet Archive!)

  • image-converter

    A useful tool for easily converting images between formats, i.e. webp to png

Tutorials

  • Flexbox Froggy

    Interactive tutorial teaching the fundamentals of CSS flexbox. I don't want to oversell it but these little frogs rocked my world while I was working on my site overhaul.

  • CSS Grid Garden

    Same as above, an interative tutorial but for CSS grid this time.

  • Flexbox reference sheet

    Quick visual reference for flexbox properties and what they do. Handy to have open while you're working when you're like "wait what's the difference between align-items and align-content again"

  • Interneting is Hard

    Extensive guide for learning HTML and CSS, structured into lessons for those who learn best with this format. I haven't gone through all of it myself but the lessons I've used to self-teach were clear and easy to follow.

  • Dannarchy's Native Neocities Hit-Counter

    I used this on the first version of lunarlily to display a visitor count and also when the site was last updated. Only works for sites hosted on Neocities, of course.

Miscellaneous

  • Marginalia Search

    An open-source search engine focused specifically on indexing non-commercial websites, it's a very fun way to explore around the indie web!

  • canine.tools

    A free collection of open-source tools and self-hosted frontends. I use their cobalt instance a lot.

  • 32-bit Cafe

    Forum full of helpful people and even more helpful resources :]

  • Fujocoded

    A cool fandom-focused collective that manages various web projects. I make use of their AO3.js library!

  • Search Engine Map

    A visual index of various search engines that shows where they source their data from. If you want to move away from Google, this can help you find an alternative.
