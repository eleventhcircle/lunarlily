tools

I use VSCode as my primary text editor. It's open-source but if you want to use it without any of Microsoft's product customization and telemetry that comes with downloading from their site, consider downloading from VSCodium instead.

I recommend installing a couple extensions to make your code writing experience more comfortable, though they're by no means required.

  • Live Server: Creates a local web server that live refreshes every time you save a file within your project
  • Beautify for VSCode: Formats your HTML, CSS, and Javascript + JSON for easier reading and editing (though you can do this without an extension too!)

My code is hosted on Github and I use deploy-to-github to manage deployments, following this tutorial to set it up.
